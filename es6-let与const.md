1. for(let i = 0; i < 3; i++)
	{
		let i = "abc";
		console.log(i)
	}
	//三次abc，
	所以这两个i的的作用域不一样，设置循环变量的为父作用域，循环体里面的为单独的子作用域
	每一次的循环，i其实都是一个新的变量。

2. const的作用域与let相同，只在声明的块级作用域里有效
   需要注意的是对于对象和数组来说，const只是保证他们分配的指针是固定的，但其指向的数据是可变的
   如：const foo = {}
   	   foo.prob = 123  这样是可以的
   	   foo = {}  这样是不可以的，因为这样改变了foo指向的地址
   	在例如：
   	   const a = []
   	   a.push("111"); 可以
   	   a.length = 0;   可以
   	   a = ["122"];   错误
